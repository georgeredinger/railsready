Rails Ready

Ruby and Ruby on Rails setup script for Linux systems

wget --no-check-certificate https://github.com/joshfng/railsready/raw/master/railsready.sh && bash railsready.sh

Distros supported:

  • Ubuntu 10.04 and 10.10
  • CentOS 5.5 (utilizes the Fedora EPEL repo)

What this gives you:

  • An updated system
  • Ruby 1.9.2 latest patch level (installed to /usr/local/bin/ruby) or RVM running 1.9.2 latest patch level
  • Imagemagick
  • All libs needed to run Rails (sqlite, mysql, etc)
  • Bundler, Passenger, and Rails gems
  • Git

Rails Ready supports a "plugin" type system. The distro is detected and a corresponding "recipe" file is pulled down to run the distro specific setup steps. Check the recipes directory to see if your distro is supported. If you would like to add support for a system fork the repo, write a recipe, and submit a pull request. Take a look at recipes/ubuntu.sh for an idea of what to model your recipe after.
